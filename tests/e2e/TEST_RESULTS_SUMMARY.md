## 技术发现

### UI 组件识别

1. **对话框**: 使用 shadcn/ui Dialog 组件
   - 遮罩层: `[data-state="open"][aria-hidden="true"]`
   - 可通过 ESC 键关闭

2. **项目选择器**: CustomSelect 组件
   - 位置: 页面头部
   - 占位符: "选择项目"
   - 值存储在 URL 参数 `?projectId=xxx`

3. **按钮文本**: 中英文混合
   - 创建按钮: "新建项目" / "Create Project"
   - 保存按钮: "创建" / "Create" / "保存" / "Save"

### 自动化测试挑战

1. **Monaco 编辑器**: 
   - 无法通过标准 Playwright 输入方法填写
   - 需要特殊处理或跳过

2. **动画延迟**:
   - 对话框打开/关闭有动画
   - 需要适当的等待时间 (1-2秒)

3. **状态同步**:
   - URL 参数变化后页面状态可能需要时间更新
   - 建议使用 `waitForSelector` 等待关键元素出现

## 首轮发现

### P0 / P1
1. 场景执行 YAML 转换不兼容旧结构
- 现象：计划执行失败，报告状态为 failed
- 根因：后端 `backend/app/api/v1/endpoints/scenarios.py` 未兼容前端旧版 `assertions/extractions` 与 `validate` 结构
- 处理：已修复并补后端单测

2. 接口定义页项目选择器截断 UUID
- 现象：选择项目后 URL 变成 `projectId=6`，列表与详情联动异常
- 根因：前端接口定义页对 UUID 使用了 `parseInt`
- 处理：已修复并补前端单测

3. 关键字编辑弹窗类型回填错误
- 现象：自定义关键字编辑时默认显示为“发送请求”
- 根因：前端把业务 `class_name` 误当成固定关键字类型
- 处理：已修复并补前端单测

4. 接口列表后端过滤不支持 UUID 项目 ID
- 现象：接口已创建但列表为空
- 根因：后端 `interfaces` 列表接口把 `project_id` 定义为 `int`
- 处理：已修复并补接口测试

5. 接口调试发送前后端契约错位
- 现象：接口定义页点击“发送”返回 500
- 根因：`/api/v1/interfaces/debug/send` 后端实现与前端当前请求体不一致
- 处理：已修复并补接口测试

### P2 / 建议项
1. 数据库配置编辑态“测试连接”要求重新输入密码，提示存在但体验略突兀
2. 场景调试页反馈偏弱，更多是 Toast 而不是结构化结果展示
3. 报告详情页当前成功报告仍可能显示“暂无步骤详情数据”

## 🚀 如何使用

### 启动服务
```bash
# 后端
cd backend
uv run uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 前端
cd frontend
npm run dev
```

### 访问应用
- 前端: http://localhost:5173
- 后端: http://localhost:8000/docs
- 登录: http://localhost:5173/login
- 项目: http://localhost:5173/projects

cd test_black/e2e

# E2E测试
npm test
